# STIWS
STIWS一款针对中药质谱解析所设计的工具，其功能包括：
1.中药成分数据库查询（来自ETCM、TCMSP、Herb及TM-MC）
2.基于中药成分的母离子匹配（包括目标药材匹配模式、混合匹配模式及全库匹配模式）
3.基于ESI-MS靶向识别特定类型的未知化合物
4.基于ESI-MS鉴定已知化合物
